stuarthadfield.com
=========

[View Site](http://stuarthadfield.com)

Building
--------

```bash
git clone git@github.com:stuarthadfield/stuarthadfield.com.git
cd stuarthadfield.com
yarn install
yarn run dev
# Open localhost:4003/index-dev.html in your browser
```

Building for Production
--------

```bash
yarn run build
```
